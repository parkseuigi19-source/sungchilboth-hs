# MVP 기능 정의서 (MVP Feature Features)

성취봇-HS의 최소 기능 제품(MVP)에 포함될 핵심 기능 정의입니다.

## 1. 공통 기능
| 기능 ID | 기능명 | 설명 | 비고 |
| :--- | :--- | :--- | :--- |
| COM-01 | 회원가입/로그인 | 학생/교사 권한 분리 및 계정 생성 | JWT 인증 |
| COM-02 | 메인 대시보드 | 권한별 맞춤형 메인 화면 진입 | |

## 2. 교사 기능 (Teacher Mode)
| 구분 | 기능 ID | 기능명 | 상세 내용 |
| :--- | :--- | :--- | :--- |
| **학급 관리** | T-CLS-01 | 학급 생성 | 학급명, 과목, 연도 설정하여 가상 교실 생성 |
| | T-CLS-02 | 학생 관리 | 학생 목록 조회 및 관리, 초대 코드 발급 |
| **평가 관리** | T-ASM-01 | 문항 출제 | 서/논술형, 단답형 문항 등록 및 성취기준 매핑 |
| | T-ASM-02 | 채점 기준 설정 | 문항별 루브릭(채점 기준) 및 배점 설정 |
| **채점/피드백** | T-GRD-01 | AI 일괄 채점 | 학생 답안에 대해 AI가 1차 채점 및 피드백 생성 |
| | T-GRD-02 | 검수 및 확정 | AI 채점 결과 수정 및 최종 점수 확정 |
| **리포트** | T-RPT-01 | 학급 리포트 | 학급 전체 성취도 분포, 취약 유형 분석 리포트(PDF) 생성 |
| | T-RPT-02 | 개별 리포트 | 특정 학생의 상세 성취 이력 조회 |

## 3. 학생 기능 (Student Mode)
| 구분 | 기능 ID | 기능명 | 상세 내용 |
| :--- | :--- | :--- | :--- |
| **학습 수행** | S-LRN-01 | 문제 풀기 | 배정된 과제/문항 풀이 및 답안 제출 |
| | S-LRN-02 | AI 튜터 | 문제 풀이 중 AI 챗봇에게 힌트 요청 및 질의응답 |
| **결과 확인** | S-RES-01 | 채점 결과 확인 | 필채점 점수 및 피드백, 해설 확인 |
| | S-RES-02 | 오답 노트 | 틀린 문항 다시 보기 및 유사 문항 추천 |
| **포트폴리오** | S-PRF-01 | 대시보드 | 나의 영역별 성취도 그래프 및 추이 확인 |
| | S-PRF-02 | PDF 다운로드 | 누적 학습 기록이 담긴 E-포트폴리오 파일 생성 및 다운로드 |

## 4. 핵심 기술 엔진 (Core Engine)
| 구분 | 설명 |
| :--- | :--- |
| **LangChain Agent** | GPT-4o 기반 교육용 에이전트. 채점 로직 및 1:1 튜터링 수행 |
| **PDF Generator** | ReportLab 라이브러리를 활용한 고품질 리포트 자동 생성 |
| **Data Analytics** | Matplotlib 활용 성취도 시각화 (레이더 차트, 추이 그래프) |
